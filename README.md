## Giới thiệu
Ứng dụng Todo List đầy đủ tính năng, hiện đại, responsive – được xây dựng hoàn toàn từ đầu bằng stack MERN.

## Tính năng chính
- Thêm / Sửa / Xóa / Hoàn thành nhiệm vụ
- Lọc theo trạng thái (Tất cả / Đang làm / Hoàn thành)
- Lọc theo ngày (Hôm nay / Tuần này / Tuần trước)
- Phân trang (Pagination) mượt mà
- Giao diện đẹp với Tailwind CSS + shadcn/ui
- Hoạt động hoàn hảo trên cả desktop & mobile
- Deploy 24/7 miễn phí: Vercel (frontend) + Render (backend)

## Công nghệ sử dụng
### Frontend
- React + Vite
- Tailwind CSS
- shadcn/ui + Lucide icons
- Axios + Zustand (state management nhẹ)
- Day.js (xử lý ngày tháng)

### Backend
- Node.js + Express
- MongoDB Atlas + Mongoose
- CORS + dotenv
- Deploy trên Render (free tier)

### DevOps & Tools
- Git + GitHub
- MongoDB Atlas (database)
## Cấu trúc thư mục
Todo-List/
├── backend/           # Express server + MongoDB
├── frontend/          # React + Vite + Tailwind
└── README.
